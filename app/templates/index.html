Upload Attendance File

How To Use

  1. Upload attendance file in the following format with exact column names:
    • Emp Id
    • Name
    • Date
    • Shift
    • From
    • To
    • First Punch
    • Last Punch
    • Total Working Hours
    • Total Break Hours
  2. On the next page, upload the salary file with the following columns:
    • Code
    • Name
    • Mobile No
    • Gender
    • Department Name
    • Designation Name
    • Salary Type
    • Salary
    • Floor
  3. If you are uploading a new salary file or for the first time, after selecting the file, click "Match All" to align employee data. You can modify the salary details after matching.
  4. If fetching attendance for another day and the salary details haven’t changed, you don’t need to upload the salary file again. Just click "Save Salary".
  5. Finally, on the report page, you can download the report as a CSV or PDF file.
